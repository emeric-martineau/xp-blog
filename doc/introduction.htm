I - Introduction

  1.1 - Qu'est-ce que XP-Blog ?

XP-Blog est un blogger utilisant le XML pour stocker ses données. Ainsi, il ne recours pas aux bases de données tel que MySQL ou PostgreSQL.

  1.2 - A qui s'adresse-t-il ?

XP-Blog s'addresse à... à ceux qui veulent faire un blog personnel (je sais, c'est un peu idiot) et qui ne veulent pas utiliser de base de données ou qui on un hébergeur de site Web qui n'a pas de base de données.
On peut biensûr détourner la fonction principale en utilisant XP-Blog comme moteur de news, mais bon le fait de recourir au XML est plus lent, donc faut pas qu'il y ait trop de visiteurs d'un coups !

  1.3 - Eléments requis

XP-Blog requière un server web (Apache de préférence puisque c'est du GPL), l'interpréteur PHP (version 3 ou 4) avec le module XML. Connaître le PHP sera un plus, mais ce n'est pas obligatoire. Et surtout des choses à dire !
