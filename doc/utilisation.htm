III - Utilisation

   3.1 - Accueil du site

Lorsqu'on arrive pour la première fois sur le site, avec le thème classic, voici ce qu'on peut obtenir :



On distingue 3 zones (plus peut-être suivant votre thème), une zone contenant les archives, une zone contenant les textes, une zone contenant la barre de navigation s'il y a trop de textes pour être affichés sur une seul page.


  3.2 - Connexion à l'administration

Pour ce connecter à l'administration du site, il faut soit exécuter le fichier auth.php ou cliquer sur le lien prévu pour cela. Nous voyons donc apparaître :
Identifiant :
Mot de passe :


Lorsque les informations d'identification sont correct, le site envoie un cookie (les bon p'tits gâteaux) contenant un numéro de session. Ce numéro généré aléatoirement, évite d'inscrire les informations d'identification.


  3.3 - Panneau d'aministration

Une fois connecté, le panneau (suivant les versions, cela peut différer) ci-dessous apparait :
 

Voir les news
Voir les brouillons
Gestion des catégories
Gestion des types
Ajouter une news
Information sur le PHP

se déconnecter


  3.4 - Déconnexion

Il est très impostant de se déconnecter, cela permet de dévalider la session en cours et si quelqu'un utilise votre ordinateur après vous, la session étant encore valide (un temps de validité trop long ?), peut se connecter sans s'identifier.


  3.5 - Gestion des textes (news, nouvelles, blog...)

    3.5.1 - Ajout d'un texte

Pour ajouter un texte, il faut déjà cliquer sur le lien dans le panneau d'adminisration. Ensuiste, on à la page suivante qui s'affiche :
Titre :
Texte :
Brouillon :
Type :
Catégorie :

Le titre et texte sont suffisament parlant pour ne pas être expliqué.
La case brouillon, indique si le texte que vous rentrez est un brouillon ou non. Si c'est un brouillon, cela signifie qu'il ne va pas apparaître sur le site mais sera tout de même sauvegarder.
Le type indique le type (je sais, je suis trop fort) de la news, par exemple : photo, poême, coup de gueule.
La catégorie... c'est la même chose que le type. En fait, les deux combinés permettre de faire un type photo avec un sous type vacances ou l'inverse. Vous pouvez l'utiliser comme bon vous semble.

    3.5.2 - Voir, éditer, supprimer une news ou un brouillon

Pour voir, éditer, supprimer une news, il suffit d'aller sur la page d'accueil du site ou de cliquer sur le lien dans le panneau d'aministration. Lorsqu'on édite une news, le date et heure ne sont pas mis à jour volontairement. L'id est un numéro unique pour identifier la news. Il est inutile de le modifier cette valeur sauf si vous savez ce que vous faites.


  3.6 - Catégories

    3.6.1 - Lister

Pour lister les catégories, il suffit de cliquer sur le lien prévu pour cela dans le panneau d'administration.

    3.6.2 - Ajouter

Pour ajouter une catégorie, il suffit de complèter le formulaire fournit avec la liste qui est celui ci dessous :

Nom :
Description :
Image :


Le nom de la catégorie est ce qui apparaîtra dans la liste ou dans les news. La description sert uniquement pour vous, pour savoir à quoi correspond la catégorie. L'image est associé à la catégorie pour faire plus jolie dans l'affichage des news. Ceci-dit nul n'empêche de ne pas l'afficher.

    3.6.3 - Supprimer

Suffit de cliquer sur le lien.

    3.6.4 - Editer

Lorsque vous éditer la catégorie, vous obtenez l'écran ci-dessous :

Id :
Titre :
Texte :
Type :
Catégorie :

L'id est un numéro unique qui identifie la catégorie. On peut changer la catérorie de toutes les news en changeant le numéro d'id. En effet, les news se réfère au numéro d'id. Cela peut être intéressant pour inverser deux catégories entre elles.


  3.7 - Types

Les types étant la même chose que les catégories (la différence se fera en fonction de ce que vous en fait) il suffit de relire la partie catégorie en remplaçant catégorie par type.
